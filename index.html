glTF Viewer Component Demo

Instructions:

  • Camera Controls: Left click and drag to rotate, right click and drag to pan, scroll to zoom
  • Load Model: Use the file picker in the top-right panel to load a glTF file
  • Mesh Visibility: Use the checkboxes in the control panel to show/hide individual meshes
  • Demo Model: The component will automatically load the existing "ms-gnk-v11.gltf" file
